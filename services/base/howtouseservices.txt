Create New Service
===================
You can create service by following steps

1) To create new service, create a folder inside services.
   Recommended is to give a name(so that it is easier to understand) which have service word inside it. Example: testservice, notifyservice, etc.
2) Create a class by extending  with BaseService
   Recommended name: Try to same this class similar to the folder name
3) We have to keep this class as Singleton so that only one instance of the service exist.
    For example check file TestService.ts

Creating Jobs
=============

1) Inside service create a folder names jobs (All the jobs will go inside this)
2) Create your job class by extending to BaseJob
 You can write your login in handle method
